1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. [Constants should be used for values that remain unchanged throughout the program. Their name reflects their purpose: they are constant and should not be modified. In contrast, variables are designed for storing data that may need to be updated or altered later. Not all values should be constants, as future changes in the code may require flexibility in data storage. ]
